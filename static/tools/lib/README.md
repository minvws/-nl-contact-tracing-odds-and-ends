# diagnosis-keys
Tools for working with Diagnosis Keys from the COVID-19 Exposure Notification / Corona-Warn-App system

## Installation
    % git clone https://github.com/mh-/diagnosis-keys
    % cd diagnosis-keys
    % pip3 install -r requirements.txt
